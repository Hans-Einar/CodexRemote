# SDP Instructions

This folder contains the general process documents for this repository's Standard Document Procedure.

Current documents:

- `SoftwareDevelopmentProcess.md`
- `HandoffProcess.md`
- `HandoffTemplate.md`

Recommended read order:

1. `AGENTS.md`
2. `SDP/AGENT-REMINDERS.md`
3. `SDP/Instructions/SoftwareDevelopmentProcess.md`
4. the active phase, sprint, or refactor documents
5. `SDP/Instructions/HandoffProcess.md` when a handoff is active or likely
6. the local `Handoff.md` inside the active work folder, if present
