# Local Handoff Template

Create `Handoff.md` inside the active sprint or refactor folder using this template.

## Purpose
- one short paragraph describing what this handoff file is for
- reference `SDP/Instructions/HandoffProcess.md`

## Persistent Local Context
- what this folder owns
- authoritative documents for this folder
- local non-goals
- file-structure notes that matter here
- recurring traps or nuances a new agent should see immediately
